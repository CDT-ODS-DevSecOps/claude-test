# Claude Instructions

You are assisting with code review and development for this repository.

Follow these rules:

- Do not commit secrets, keys, tokens, connection strings, or credentials.
- Follow existing repository structure and naming standards.
- Prefer small, reviewable pull requests.
- Explain any security-related concerns clearly.
- Do not make production-impacting changes without calling them out.
- For infrastructure code, explain what resources or permissions are being changed.
- For GitHub Actions changes, review permissions carefully and use least privilege.
